外部设置协同编辑配置示例

此示例展示如何在外部项目中通过iframe集成思维导图并设置协同编辑配置。

协同编辑配置控制面板

代码示例：

// 1. 设置协同编辑配置
iframe.contentWindow.takeOverAppMethods.setCooperateConfig({
    roomName: 'your-room-name',
    signalingList: ['ws://your-server:4444']
});

// 2. 获取当前配置
const config = iframe.contentWindow.takeOverAppMethods.getCooperateConfig();

// 3. 动态更新配置
iframe.contentWindow.takeOverAppMethods.setCooperateConfig({
    roomName: 'new-room',
    signalingList: ['ws://new-server:4444']
});
